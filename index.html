Applications Web et Bases de données

Exercices module 01

Pour tous les exercices, vous devez écrire l'algorithme qui effectue les tâches demandées. Le contenu des variables est appelé à changer.

Question 1

Considérant que les notes dans la variable etudiant sont sur 10, retournez dans un tableau de chaines (string) chaque nom suivi de sa note sur 100.

Question 2

Retournez dans un tableau les noms des étudiants selon l'ordre croissant des matricules.

Question 3

Retournez dans un tableau les noms des étudiants en échec.

Question 4

Calculez la moyenne du groupe sur 100.

Question 5

Calculez l'écart type du groupe du 100.

Question 6

Retournez dans un tableau le nom complet des étudiants dont le nom de famille est plus petit ou égal à 6 caractères.

Question 7

Retournez dans un tableau tous les noms de étudiants, où chaque nom est précédé et suivi de 3 étoiles (ex.: *** Adam Bernard ***).

Question 8

À partir du tableau défini dans la fonction Question8(), retournez un tableau sans les doublons.

Question 9

À partir du tableau défini dans la fonction Question9(), retournez la deuxième plus grande valeur.

Question 10

À partir des deux tableaux définis dans la fonction Question10(), Retournez la paire dont la somme se rapproche le plus de la cible. Vous devez former une paire en prenant un nombre dans chaque tableau.
